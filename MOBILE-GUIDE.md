## Using the Mobile App

### First Time Setup

1. **Open the installed app** from your home screen
2. The app will load with a futuristic loading animation
3. You'll see the Dashboard with 4 main metrics
4. Start by adding your first habit or task!

### Adding a Habit with Specific Time

1. Go to **"Habits"** from sidebar
2. Tap **"+ New Habit"**
3. Enter habit name (e.g., "Morning Exercise")
4. Choose an icon
5. Select frequency (Daily/Weekdays/Weekends)
6. **Set specific time**: Tap the time field and pick exact time (e.g., 06:30)
7. Choose time of day category
8. Tap **"Save Habit"**

Your habit will now show with a ‚è∞ clock icon and the time!

### Setting Up Tasks

1. Tap **"Smart Tasks"** in sidebar
2. Tap **"+ New Task"**
3. Fill in:
   - Title
   - Priority (High/Medium/Low)
   - Category
   - Deadline (optional)
   - Description
4. Save

AI will automatically prioritize based on urgency!

### Using Focus Timer

1. Go to **"Focus Score"**
2. Tap **"Start Focus"**
3. Timer starts countdown (default 25 minutes)
4. Focus on your work!
5. Take a break when timer completes

## Troubleshooting

### App Won't Install

**Android:**

- Make sure you're using Chrome, Edge, or Samsung Internet
- Enable installation from unknown sources (if needed)
- Check if you have storage space

**iOS:**

- **Must use Safari browser** (not Chrome/Firefox)
- Update iOS to 14.3 or later
- Try airplane mode ON then OFF

### Data Not Saving

1. Check if LocalStorage is enabled in browser
2. Don't use incognito/private mode
3. Clear browser cache and reinstall
4. Make sure you have storage space

### App Looks Wrong

1. Force-close and reopen app
2. Clear cache:
   - Android: Settings ‚Üí Apps ‚Üí AI-POS ‚Üí Clear Cache
   - iOS: Settings ‚Üí Safari ‚Üí Clear History and Data (then reinstall)
3. Update browser to latest version

### Install Banner Not Showing

1. Wait 2-3 minutes on the page
2. Use browser menu instead (Method 2)
3. Check browser version (update if old)
